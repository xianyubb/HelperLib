Class LLSE_Device

📱 设备信息对象

在LLSE中，使用「设备信息对象」来操作和获取某一个玩家使用的游戏设备的相关信息。

该类没有构造函数，请使用Player.getDevice()创建

注意：不要长期保存一个设备信息对象
当设备对应的玩家退出游戏时，对应的对象将变得无效。
因此，如果有长期操作某个对象的需要，请通过上述途径获取实时的设备信息对象
