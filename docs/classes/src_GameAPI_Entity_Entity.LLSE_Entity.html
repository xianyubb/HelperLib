Class LLSE_Entity

🎈 实体对象

在脚本引擎中，使用「实体对象」来操作和获取某一个实体的相关信息。

该类没有构造函数，请使用下面的方法获取该类对象

  • 从监听事件中获取
  • mc.getAllEntities()
  • mc.spawnMob()
  • mc.cloneMob()
  • 从其他API中获取

注意：不要长期保存一个实体对象
当实体对象对应的实体被销毁时，对应的实体对象将同时释放。
因此，如果有长期操作某个实体的需要，请通过上述途径获取实时的实体对象
