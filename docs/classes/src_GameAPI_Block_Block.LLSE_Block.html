Class LLSE_Block

📦 方块对象

在LLSE中，使用「方块对象」来操作和获取某一类方块的相关信息

该类没有构造函数，请通过其他方式获得类对象

注意：不要长期保存一个方块对象
当方块对象对应的方块被销毁时，对应的方块对象将变得无效。
因此，如果有长期操作某个方块的需要，请通过事件提供的参数或者使用mc.getBlock()获取实时的方块对象
