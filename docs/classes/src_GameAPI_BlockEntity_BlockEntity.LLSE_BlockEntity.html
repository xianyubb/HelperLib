Class LLSE_BlockEntity

方块实体对象

在LLSE中，使用「方块实体对象」来操作和获取与特定方块关联的附加数据

该类没有构造函数，请通过其他方式获得类对象

注意：方块实体对象 不是一种 实体！他们之间并没有特别的关系

注意：不要长期保存一个方块实体对象
当方块对象对应的方块被销毁时，对应的方块实体对象将变得无效。
因此，如果有长期操作某个方块实体的需要，请通过Block.getBlockEntity()获取实时的方块实体对象
